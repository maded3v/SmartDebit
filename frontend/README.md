# SmartDebit Frontend

Интерфейсная часть учебного кейса SmartDebit для банковского кабинета.

## Что реализовано

- Страницы: `Главная`, `Обзор карты`, `Операции` (со встроенным SmartDebit), `Личный кабинет`.
- Верхняя навигация и карточный интерфейс в стиле банковского кабинета Т-Банка.
- SmartDebit внутри страницы операций: тумблер, критичные алерты, список платежей, статусы.
- Модалка управления подпиской (`пропустить месяц`, `отключить/включить`).
- Форма ручного добавления регулярного платежа.
- Аналитический donut-график и блок уведомлений.

## Запуск

```bash
npm install
npm run dev
```

По умолчанию фронт ожидает API на `http://localhost:4000/api/v1`.
Если нужно изменить URL, создайте `.env` на базе `.env.example`.

## Авторизация (frontend)

- `VITE_AUTH_PROVIDER=mock` — локальный мок-режим (по умолчанию).
- `VITE_AUTH_PROVIDER=backend` — вызовы `/auth/session/`, `/auth/login/`, `/auth/register/`, `/auth/logout/`.

## Тесты
